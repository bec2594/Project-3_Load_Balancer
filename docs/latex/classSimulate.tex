\doxysubsection{Detailed Description}
High-\/level simulation controller that owns the main components and drives the simulation. \mbox{[}15\mbox{]}. 

\mbox{\hyperlink{classSimulate}{Simulate}} is responsible for\+:
\begin{DoxyItemize}
\item Loading configuration values (\mbox{\hyperlink{classConfig}{Config}}) \mbox{[}15\mbox{]}
\item Creating the \mbox{\hyperlink{classRequestGenerator}{Request\+Generator}}, \mbox{\hyperlink{classLogFileStats}{Log\+File\+Stats}} collector, and main \mbox{\hyperlink{classLoadBalancer}{Load\+Balancer}} \mbox{[}14\mbox{]}\mbox{[}15\mbox{]}
\item Seeding initial conditions (blocked IP ranges and initial queue fill) \mbox{[}14\mbox{]}\mbox{[}15\mbox{]}
\item Running the simulation loop for \mbox{\hyperlink{classConfig_a521996e25cf63996d543463356de4c66}{Config\+::duration\+\_\+cycles}} ticks \mbox{[}14\mbox{]}\mbox{[}15\mbox{]}
\item Writing events to a global log file and printing a final summary \mbox{[}14\mbox{]}\mbox{[}15\mbox{]} 
\end{DoxyItemize}
