{\footnotesize\ttfamily void Load\+Balancer\+::manage\+\_\+server\+\_\+count (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Adjusts the number of Web\+Servers based on current queue size thresholds. \mbox{[}6\mbox{]}. 

Adds or removes servers based on queue size thresholds, at configured intervals. \mbox{[}5\mbox{]}.

Queue thresholds are scaled by the current server count (see \mbox{\hyperlink{classConfig}{Config}} values). \mbox{[}6\mbox{]}

This method only performs scaling checks if enough time has elapsed since the last check (server\+\_\+check\+\_\+interval\+\_\+cycles). \mbox{[}5\mbox{]}

Thresholds are computed as\+:
\begin{DoxyItemize}
\item upper = queue\+\_\+high\+\_\+count $\ast$ current\+\_\+server\+\_\+count
\item lower = queue\+\_\+low\+\_\+count $\ast$ current\+\_\+server\+\_\+count
\end{DoxyItemize}

If current queue size is below lower, an idle server may be removed. If current queue size is above upper, a server is added. \mbox{[}5\mbox{]} \mbox{\Hypertarget{classLoadBalancer_a4579c1d098ac5666d750c93c87a3125c}\label{classLoadBalancer_a4579c1d098ac5666d750c93c87a3125c}}
