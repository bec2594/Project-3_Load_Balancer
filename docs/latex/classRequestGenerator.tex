\doxysubsection{Detailed Description}
Generates randomized \mbox{\hyperlink{classRequest}{Request}} instances using simulation configuration parameters. \mbox{[}13\mbox{]}. 

Uses a Mersenne Twister RNG and several distributions to generate\+:
\begin{DoxyItemize}
\item IPv4 addresses (from a set of private ranges)
\item Processing times within the configured min/max bounds
\item Job types (\textquotesingle{}P\textquotesingle{} or \textquotesingle{}S\textquotesingle{}) \mbox{[}13\mbox{]} 
\end{DoxyItemize}
