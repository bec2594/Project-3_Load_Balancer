\doxysubsection{Detailed Description}
Aggregates counters and timing metrics for the simulation and produces a summary report. \mbox{[}8\mbox{]}. 

This class tracks\+:
\begin{DoxyItemize}
\item Total processed and blocked requests
\item Server scaling operations (added/removed)
\item Queue statistics (max and average sampling)
\item Wait time and processing time for completed requests
\item Optional final snapshot values (queue size, active/inactive servers) \mbox{[}8\mbox{]} 
\end{DoxyItemize}
