\doxysubsection{Detailed Description}
Represents a single server that can process at most one \mbox{\hyperlink{classRequest}{Request}} at a time. \mbox{[}17\mbox{]}. 

A \mbox{\hyperlink{classWebServer}{Web\+Server}} tracks the currently assigned request (if any), the remaining processing cycles, and whether it is busy. It provides methods to accept a request, advance time, query availability, and return a completed request. \mbox{[}17\mbox{]}
