\doxysubsection{Detailed Description}
Holds configuration parameters for the load balancer simulation. 

This class provides default values (via the constructor) and supports loading overrides from a colon-\/separated configuration file. See \mbox{\hyperlink{classConfig_aa9af335c875d49cecbab1dc67b001727}{load\+\_\+from\+\_\+file()}}.
