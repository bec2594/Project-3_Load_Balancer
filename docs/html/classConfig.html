Detailed Description

Holds configuration parameters for the load balancer simulation.

This class provides default values (via the constructor) and supports loading overrides from a colon-separated configuration file. See load_from_file().

◆ load_from_file()

bool Config::load_from_file ( const std::string &  config_file)

Loads configuration parameters from a file.

Loads configuration parameters from a colon-separated configuration file.

The file is expected to contain lines formatted as:

key: value

Empty lines and lines beginning with '#' are ignored.

Recognized keys include (as implemented): initial_server_count, queue_low_count, queue_high_count, initial_queue_fill, min_process_time, max_process_time, duration_cycles, server_check_interval_cycles, new_request_probability, log_file_name [1].

Parameters
config_filePath to the configuration file.
Returns
true if the file opens successfully and is parsed; false otherwise.

The loader:

  • Opens the file at config_file
  • Ignores empty lines and lines beginning with '#'
  • Expects "key: value" pairs
  • Trims whitespace from keys/values
  • Assigns recognized keys to member variables
  • Prints a warning for unknown keys
Parameters
config_filePath to the configuration file to read.
Returns
true if the file could be opened and parsed; false if the file could not be opened.
