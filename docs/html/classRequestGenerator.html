Detailed Description

Generates randomized Request instances using simulation configuration parameters. [13].

Uses a Mersenne Twister RNG and several distributions to generate:

  • IPv4 addresses (from a set of private ranges)
  • Processing times within the configured min/max bounds
  • Job types ('P' or 'S') [13]

Generates a randomized private IPv4 address string. [12].

Returns
IPv4 address in dotted-decimal notation.

Randomly chooses one of three private IP ranges and then fills the remaining octets randomly:

  • choice 0: "192.168.X.Y"
  • choice 1: "10.X.Y.Z"
  • choice 2: "172.16.X.Y" (note: this implementation uses 172.16 specifically) [12]
