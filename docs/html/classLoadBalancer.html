◆ manage_server_count()

void LoadBalancer::manage_server_count ( )

Adjusts the number of WebServers based on current queue size thresholds. [6].

Adds or removes servers based on queue size thresholds, at configured intervals. [5].

Queue thresholds are scaled by the current server count (see Config values). [6]

This method only performs scaling checks if enough time has elapsed since the last check (server_check_interval_cycles). [5]

Thresholds are computed as:

  • upper = queue_high_count * current_server_count
  • lower = queue_low_count * current_server_count

If current queue size is below lower, an idle server may be removed. If current queue size is above upper, a server is added. [5]
