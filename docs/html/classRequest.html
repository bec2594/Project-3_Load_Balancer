Constructs a Request with the provided identifiers, IPs, processing time, type, and arrival time. [10].

The constructor initializes:

  • id with request_id
  • IP_in/IP_out using move semantics to avoid extra string copies
  • processing_time and job_type
  • arrival_time
  • start_time and completion_time to -1 (meaning "not yet started/completed") [10]

void Request::print ( std::ostream &  os) const

Prints a human-readable representation of the request to an output stream. [10].

Always prints the basic request fields (id, IPs, processing time, type, arrival time). If the request has started, prints start_time. If the request has completed, prints completion_time and derived metrics:

  • Total System Time = completion_time - arrival_time
  • Wait Time = start_time - arrival_time [10]
Parameters
osOutput stream to write to (e.g., std::cout or a file stream). [10]
